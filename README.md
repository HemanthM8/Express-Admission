#  Express-Admission

A simple admissions portal built with Express.js that serves a responsive HTML form using EJS and handles student applications in memory.

## Project Structure

```
Express-Admission/
├── views/
│ ├── admission.ejs
│ └── confirmation.ejs
├── public/
│ └── style.css
├── app.js
├── package.json
├── package-lock.json
```
## Installation

```bash
npm install
```

## To run
```bash
node app.js
```
